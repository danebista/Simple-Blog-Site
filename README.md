## Technology Used:

1. FrontEnd : Vanilla JS
2. Backend: Node JS
3. No database, No hosted server, stores data to local file.

## How to setup:

#### In the cmd

1. Inside the node folder ```` npm install ```
2. ``` npm run dev ```
3. Open index.html on the frontend folder
